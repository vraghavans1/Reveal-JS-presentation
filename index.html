Reproducible Metrics (Code)

from math import pow

# Simple financial helpers for the demo

def cagr(begin, end, years):
    return pow(end / begin, 1/years) - 1

def roe(net_income, equity):
    return net_income / equity

print("CAGR: ", round(cagr(950, 1240, 3) * 100, 2), "%")
print("ROE:  ", round(roe(246, 1680) * 100, 2), "%")

Financial Formulas (MathJax)

Net Present Value (NPV): \( \mathrm{NPV} = \sum_{t=0}^{T} \frac{C_t}{(1+r)^t} \)

Weighted Average Cost of Capital (WACC): \( \mathrm{WACC} = \tfrac{E}{V} r_e + \tfrac{D}{V} r_d (1-\tau) \)

Earnings Per Share (EPS): \( \mathrm{EPS} = \tfrac{\text{Net Income} - \text{Dividends}}{\text{Weighted Avg. Shares}} \)
